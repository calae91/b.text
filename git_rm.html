la commande git rm

le git rm permet de supprimer les fichiers de l'arbre de travail et de l'indice.

Syntaxe

- git rm

Utlisation

rm 'fichier.html'

- git rm -f

Force la suppression d'un dossier .

Affichage

rm 'titi.txt'
C:\Users\rom.PC-de-BELLEVUE\Documents\GitHub\abc [topic +0 ~0 -1]>

- git rm -n
Montre ce que la commande fait mais ne l'applique pas.

Affichage

rm 't.txt'
user@lubuntu-13:~/Bureau$ ls
Documents t.txt
